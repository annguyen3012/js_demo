- getElementsByClassName: Get all elements with the specified class name: document. var x = document.getElementsByClassName("example"); - getElementsByTagName: Get all elements with the specified tag name. var x = document.getElementsByTagName("li"); - querySelector: Get the first element in the document with specified class name: var x = document.querySelector("example"); - querySelectorAll: Get all elements in the document with specified class name: var x = document.querySelectorAll("example");
